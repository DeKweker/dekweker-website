## Mappen
- clients/      = echte klantprojecten
- personal/     = je eigen projecten
- templates/    = herbruikbare startpunten
- experiments/  = tests en prototypes
- archive/      = afgeronde projecten (read-only)
- _docs/        = checklists, offertes, notes
- _snippets/    = herbruikbare codeblokken
- _scripts/     = bash scripts voor deploy/backup/controle
- _assets/      = algemene fonts/icons (niet project-specifiek)

## Regel
Project-assets die effectief op de site moeten: altijd in het project zelf (assets/).
